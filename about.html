About This Project

Book Mood Selector is a fun, interactive web app that recommends books based on how you're feeling. Whether you're feeling adventurous, nostalgic, romantic, or a little eerie — there's always a perfect book match waiting for you.

👩‍💻 Built By

Created with care by Alessandra Pastor, a passionate student-athlete with a love for literature, tech, and track. This project was built using HTML, CSS, and JavaScript — and designed to be lightweight, responsive, and fun!

💡 How It Works

The app uses JavaScript to read your selected mood (or quiz answers), then dynamically displays a list of curated book recommendations based on that mood.

🌱 What's Next?

We’re building a community! Soon, users will be able to share their mood and discover books through a public forum. Stay tuned!

🧰 Tech Stack

  • HTML5, CSS3, JavaScript
  • LocalStorage for forum features
  • Google Fonts, Emoji UX
